Byte Converter is a simple utility to convert numeric units of digital information from one form to another. 
The supported units are,

Decimal: Bits, Bytes, KiloBytes, MegaBytes, GigaBytes, TeraBytes, PetaBytes, ExaBytes, ZettaBytes, YottaBytes.
Binary: Bits, Bytes, KibiBytes, MebiBytes, GibiBytes, TebiBytes, PebiBytes, ExbiBytes, ZebiBytes, YobiBytes.

Requirements: .NET Framework (2.0) or above
Download the executable file at: https://github.com/akmamansoor/ByteConverter/releases
